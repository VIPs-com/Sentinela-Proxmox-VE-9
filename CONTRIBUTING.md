# Contribuir — Sentinela Proxmox

Obrigado por considerar melhorar o **Sentinela Proxmox** (release canônica v1.0).

## Alterações técnicas

- Cite sempre uma **fonte oficial** (wiki Proxmox, Debian, Tailscale, CrowdSec, OpenSSH, etc.) no PR ou issue.
- O guia canônico é [docs/sentinela-proxmox-v1.0.md](docs/sentinela-proxmox-v1.0.md). Atualize [docs/audit-matrix.md](docs/audit-matrix.md) quando o alinhamento de uma fase mudar.
- Histórico editorial do repositório: [docs/interno/CHANGELOG-repositorio.md](docs/interno/CHANGELOG-repositorio.md).

## Reportar comando desatualizado

- Issue com: `pveversion`, saída do erro, link da documentação oficial atual.
- Patch pequeno preferível a issue longa.

## Estilo

- Mantém o tom do guia (português do autor; consistência PT-BR/PT onde já existir).
- Evita duplicar blocos longos — link para a seção canônica no guia.
